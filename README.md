## Simple MERN

Simple MERN is Social network developers

```
Authentification
Profile
Posts
```

## Architecture

Mongo DB
Express JS
React JS
Node JS

## Library Used

- bcryptjs
- body-parser
- express
- jsonwebtoken
- mongoose
- passport
- passport-jwt
- validator

## Running

- Clone this repository
- Setup mongoURI in ./config/keys.js
- composer install
- npm run server
- Open your browser and type http://localhost:5000
